# 02. Numbers

## 1. 수학 연산자

| 연산자  |      설명       |     예     |  결과  |
| :--: | :-----------: | :-------: | :--: |
|  +   |     plus      | 215 + 313 | 528  |
|  -   |     minus     |   5 - 3   |  2   |
|  *   |   multiply    |   3 * 5   |  15  |
|  /   |   division    |   6 / 4   | 1.5  |
|  //  | floor divison |  6 // 4   |  1   |
|  %   |   reminder    |   6 % 4   |  2   |
|  **  |     power     |  2 ** 3   |  8   |

<br>

## 2. 산술 연산자의 결합

```
a = 100
result = a - 3
```

위의 연산자는

```
a = 100
a -= 3
```

으로 줄여 쓸수 있다.

<br>

## 4. 진수 (base)

사람들은 기본적으로 *10진수*를 많이 사용하지만, python에서는 **2진수**, **8진수**, **16진수**를 표현할 수 잇다.

- 2진수(binary) : 0b, 0B
- 8진수(octal) : 0o, 0O
- 16진수(hex) : 0x, 0X

<br>
